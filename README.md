# Delhi Tourist Spots Web App

A web application showcasing various tourist destinations in Delhi using a JSON server for data, React for the frontend, and Bootstrap for styling.

---

## Features

- Displays a list of popular tourist spots in Delhi.
- View details of each spot with an image and description.
- Interactive image gallery for browsing.

---

## Technologies Used

- **Frontend**: React, React Router
- **Styling**: Bootstrap
- **Backend**: JSON Server
- **Language**: JavaScript (ES6+)
